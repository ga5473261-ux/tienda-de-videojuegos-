La Tienda de Videojuegos es un proyecto desarrollado en Rust con Anchor sobre la blockchain de Solana, cuyo objetivo es ofrecer una plataforma descentralizada para la gestión de videojuegos digitales. El sistema permite que un propietario cree su tienda, registre títulos con información como nombre, plataforma y precio, y administre su disponibilidad.
El contrato inteligente asegura que solo el dueño de la tienda pueda realizar operaciones críticas como agregar, eliminar o modificar juegos, garantizando seguridad y control sobre el inventario. Además, incluye funciones para consultar la lista completa de títulos y obtener el número total de videojuegos registrados.
En esencia, este proyecto combina la transparencia de la blockchain con una estructura modular y escalable, brindando una solución práctica para administrar un catálogo digital de videojuegos de manera confiable y accesible.
